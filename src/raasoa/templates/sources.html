Available Connectors

SharePoint

Microsoft SharePoint Online

Sync documents from SharePoint libraries. Configure a Power Automate flow or Azure Logic App to send webhook events when documents change.

Setup Instructions

1. In Power Automate, create a new flow triggered by "When a file is created or modified" in your SharePoint library.

2. Add an HTTP action that sends a POST request to:

{{ webhook_url }}/v1/webhooks/ingest

3. Set the request body:

{
  "event": "document.created",
  "source": "sharepoint",
  "title": "@{triggerOutputs()?['body/Name']}",
  "content": "@{body('Get_file_content')}",
  "source_object_id": "@{triggerOutputs()?['body/Id']}",
  "source_url": "@{triggerOutputs()?['body/Link']}"
}

4. Add header: X-Tenant-Id: 00000000-0000-0000-0000-000000000001

Jira

Atlassian Jira Cloud

Index Jira tickets, epics, and project documentation. Use Jira Automation rules to push issue updates to RAASOA.

Setup Instructions

1. In Jira, go to Project Settings → Automation → Create Rule.

2. Trigger: "Issue created" or "Issue updated".

3. Action: "Send web request" to:

{{ webhook_url }}/v1/webhooks/ingest

4. Body (JSON):

{% raw %}{
  "event": "document.updated",
  "source": "jira",
  "title": "{{issue.key}}: {{issue.summary}}",
  "content": "{{issue.description}}",
  "source_object_id": "{{issue.key}}",
  "source_url": "{{issue.url}}"
}{% endraw %}

Confluence

Atlassian Confluence Cloud

Sync wiki pages and documentation. Use Confluence Automation to push page changes to RAASOA.

Setup Instructions

1. In Confluence, go to Space Settings → Automation.

2. Trigger: "Page published" or "Page updated".

3. Action: "Send web request" to:

{{ webhook_url }}/v1/webhooks/ingest

4. Body (JSON):

{% raw %}{
  "event": "document.updated",
  "source": "confluence",
  "title": "{{page.title}}",
  "content": "{{page.body}}",
  "source_object_id": "{{page.id}}",
  "source_url": "{{page.url}}"
}{% endraw %}

Custom Webhook

Any HTTP-capable system

Connect any system that can send HTTP requests. Send document events to the webhook endpoint.

Webhook Endpoint

Endpoint:

POST {{ webhook_url }}/v1/webhooks/ingest

Events: document.created, document.updated, document.deleted

Example:

curl -X POST {{ webhook_url }}/v1/webhooks/ingest \
  -H "Content-Type: application/json" \
  -H "X-Tenant-Id: {{ tenant_id }}" \
  -d '{
    "event": "document.created",
    "source": "custom",
    "title": "My Document",
    "content": "Document text content...",
    "source_object_id": "unique-id-123"
  }'

Batch Import

Import an entire directory of files at once using the CLI: 

# Import all files from a directory
uv run python -m raasoa.worker batch ingest /path/to/documents/

# Or use the Python SDK
from raasoa_client import RAGClient
client = RAGClient("{{ webhook_url }}")
for file in Path("docs/").glob("**/*.pdf"):
    client.ingest(file)
